Vibe-Coding Integrations (Top 10)

Copy these settings into your favorite coding GUI tools to use proXiPT as a local AI endpoint.

1. Cursor IDE

1. Open Cursor Settings > Models.
2. Add a new OpenAI-compatible Base URL.
Base URL: http://localhost:8787/v1
API Key: dummy
3. Add Custom Model Names (e.g., auto, claude-3.5-sonnet).
4. Toggle them on to use them in chat/composer.

2. Cline (Roo Code) - VS Code

1. Open VSCode Settings for the Extension.
2. Select API Provider: OpenAI Compatible.
Base URL: http://localhost:8787/v1
API Key: dummy
Model ID: auto

3. Continue.dev (VS Code / JetBrains)

1. Open your config.json setup file.
2. Add this block to your "models" array:
{
  "title": "proXiPT",
  "provider": "openai",
  "model": "auto",
  "apiKey": "dummy",
  "apiBase": "http://localhost:8787/v1"
}

4. Aider (Terminal Pair Programmer)

Start the CLI tool with these specific flags:
aider --openai-api-base http://localhost:8787/v1 --openai-api-key dummy --model openai/auto

5. OpenHands (Agent)

Define the local connection directly in your workspace .env file:
LLM_PROVIDER="openai"
LLM_BASE_URL="http://localhost:8787/v1"
LLM_API_KEY="dummy"
LLM_MODEL="openai/auto"

6. OpenClaw (Autonomous OS Agent)

Configure the custom provider in your `~/.openclaw/openclaw.json`:
"models": {
  "providers": {
    "proxipt": {
      "base_url": "http://localhost:8787/v1",
      "api": "openai-completions",
      "api_key": "dummy"
    }
  }
}

7. Zed IDE

Open your settings.json via cmd+, and configure:
"language_models": {
  "openai": {
    "api_url": "http://localhost:8787/v1",
    "available_models": [{"name": "auto", "max_tokens": 128000}]
  }
}

8. CodeGPT (VS Code)

1. Provider: OpenAI.
2. Note: CodeGPT requires the exact completions path:
Custom Endpoint: http://localhost:8787/v1/chat/completions
Model: auto
Auth Token: dummy

9. ClawCode (Harness CLI)

Configure it via environment variables before launching your workspace:
export OPENAI_BASE_URL="http://localhost:8787/v1"
export OPENAI_API_KEY="dummy"
clawcode --model auto

10. AutoGPT

Route the Python agent loop through your local endpoint in `.env`:
OPENAI_API_BASE_URL="http://localhost:8787/v1"
OPENAI_API_KEY="dummy"
SMART_LLM="auto"
FAST_LLM="auto"

11. Claude Code (Anthropic CLI)

1. Claude Code only speaks Anthropic API format. To bridge it to ProxiPT, install litellm:
pip install litellm
2. Start LiteLLM proxy to translate Anthropic requests to ProxiPT's OpenAI format:
litellm --model openai/auto --api_base http://localhost:8787/v1 --api_key dummy
3. Run Claude Code pointing to the LiteLLM proxy port (usually 4000):
export ANTHROPIC_BASE_URL="http://localhost:4000"
export ANTHROPIC_API_KEY="dummy"
claude
